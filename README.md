<h1>📊 Machine Learning Projects Overview</h1>

<h2>A) Numerical Dataset</h2>
<h3>1️⃣ Healthcare Insurance Expenses Prediction</h3>
<p><strong>Goal:</strong> Develop predictive models to estimate healthcare expenses using demographic and health features.</p>
<p><strong>Dataset:</strong> <a href="https://www.kaggle.com/datasets/arunjangir245/healthcare-insurance-expenses/data" target="_blank">Healthcare Insurance Expenses</a></p>

<p><strong>Models Implemented:</strong></p>
<ul>
    <li>Linear Regression</li>
    <li>K-Nearest Neighbors (KNN)</li>
</ul>

<p><strong>Key Features:</strong> age, sex, BMI, number of children, smoker status, region</p>
<p><strong>Target:</strong> charges (insurance expenses)</p>

<h2>B) Image Dataset</h2>
<h3>2️⃣ Fashion Item Classification</h3>
<p><strong>Goal:</strong> Build a model to classify fashion items such as clothing and accessories.</p>
<p><strong>Dataset:</strong> Fashion MNIST (60,000 training images, 10,000 test images, 28x28 grayscale)</p>

<h3>Models Implemented:</h3>
<ul>
    <li>Logistic Regression</li>
    <li>K-Means Clustering (for unsupervised grouping)</li>
</ul>
